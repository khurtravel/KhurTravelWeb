This page demonstrates the HTML structure before transformation to clean URLs.

Notice the following:

  • All links have .html extensions
  • CSS and JS references use relative paths
  • The page is in the root directory

After running the transformation script:

  • This file will be moved to /example-page/index.html
  • All links will be updated to use directory style (e.g., /about-us/ instead of about-us.html)
  • Resource paths will be made absolute (e.g., /css/main.css instead of css/main.css)
